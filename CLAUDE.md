# Rock'n Wok 프로젝트 설정

## 별명
- 사용자는 Claude Code를 **랄프(Ralf)**라고 부릅니다.

## 프로젝트 개요
- Korean Fusion Dining 레스토랑 랜딩페이지
- 정적 사이트 (HTML/CSS/JS, 프레임워크 없음)
- 영문(index.html) 메인, 한국어(ko.html) 서브

## 배포
- GitHub: https://github.com/ondallio/rocknwok
- Vercel: https://rocknwok.vercel.app
- GitHub Pages: https://ondallio.github.io/rocknwok/

## 디자인
- 다크 테마: #0F0F0F 배경, 그라데이션 적용
- 액센트: #E8632B (Burnt Orange)
- 폰트: DM Serif Display (히어로 타이틀), Pretendard (한글 타이틀), Manrope (본문)
- 디자인 파일: pencil-new.pen

## 세션 시작 시 읽을 파일
1. `CLAUDE.md` - 프로젝트 설정 (자동 로드)
2. `WORK_LOG.md` - 작업 이력 및 에러 기록
3. `plan.md` - 기획안 (원본 컨셉, 섹션 구조)
4. `index.html` - 현재 메인 페이지 (영문)
5. `styles.css` - CSS 변수, 레이아웃 구조 파악
